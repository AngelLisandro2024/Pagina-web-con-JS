# 📰 📰 Generador de Noticias Clickbait (JS Dynamic Web)

Este proyecto es una aplicación web interactiva que utiliza **JavaScript Vanilla** para generar dinámicamente una interfaz de noticias con titulares aleatorios. Es un ejercicio avanzado de manipulación del DOM y lógica de programación.

## 🚀 Funcionalidades Técnicas
* **Generación Dinámica:** Uso de bucles `for` para renderizar contenido HTML desde el script.
* **Algoritmo de Aleatoriedad:** Sistema de arrays y funciones matemáticas (`Math.random`) para crear combinaciones únicas de titulares y asignación de imágenes.
* **Diseño Responsivo:** Implementación de **Bootstrap 5** con un sistema de rejilla (Grid) adaptable a móviles, tablets y escritorio.
* **Estilos Personalizados:** Uso de CSS3 para sobrescribir componentes de Bootstrap y mejorar la identidad visual.

---

## 🛠️ Tecnologías Utilizadas
* **Lenguajes:** HTML5, CSS3, JavaScript (ES6+).
* **Framework:** Bootstrap 5.
* **Lógica:** Inyección dinámica de nodos en el DOM.
